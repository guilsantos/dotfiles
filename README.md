# dotfiles - Mac/Linux
> Repositório concentra todos os arquivos de configuração necessário para o start-up da máquina desenvolvimento.

## Instalação

```sh
curl -L https://raw.githubusercontent.com/guilhermegregio/dotfiles/master/install.sh | bash
```

## Desenvolvimento
